# Tic-Tac-Toe Game

## Description
This project is a simple implementation of the classic Tic-Tac-Toe game using HTML, CSS, and JavaScript. It allows two players to take turns marking spaces in a 3x3 grid, aiming to get three marks in a row, column, or diagonal.

## Deployment
The game is deployed at: [Tic-Tac-Toe Game](https://rishabh022.github.io/Tic-Tac-Toe/)

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Two-player gameplay
- Interactive UI with clear game status
- Responsive design for different screen sizes
